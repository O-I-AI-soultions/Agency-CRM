# Website Templates (Web Agency Track)

Reusable site skeletons — the agency's products. Build once, customize per client from a `client.json` config file. Not started yet; pipeline track comes first.

Full plan: [docs/business-plan.md](../docs/business-plan.md)

## Planned templates

1. **Landing Page** — single page: hero, services, about, gallery, testimonials, contact + map. All content driven by `client.json`.
2. **Booking & Customer Management** — landing page + appointment booking (Calendly embed or Google Sheets backend) + simple admin view.
3. **Service Payments** — landing page + pay-now / deposit page (Stripe / Apple Pay embeds, no backend).
